ESSIONAL PROMPT FOR REPLIT

:::writing{variant="standard" id="48361"}
Build a Professional Personal Finance Web App called My Daily Budget.

The app should be clean, modern, mobile-first, and production ready.

Current features already exist:
- Login with Google
- Dashboard
- Add Income
- Add Expense
- Categories
- Reports
- Profile & Settings
- Export to Excel / PDF

Now upgrade the app into a full professional finance tracker with the following features.

--------------------------------

GENERAL REQUIREMENTS

- Mobile-first responsive design
- Modern UI with smooth animations
- Clean cards and soft gradients
- Fast loading
- Offline support
- Dark mode support
- Progressive Web App (PWA)

Use:
- Clean modular code
- Reusable components
- Error handling
- Loading states
- Empty states

--------------------------------

1. DASHBOARD IMPROVEMENTS

Dashboard should show:

• Current Balance  
• Total Income (Month)  
• Total Expense (Month)  
• Remaining Budget  
• Today's Spending  
• Weekly Spending  
• Top Spending Category  

Charts:

• Pie Chart for category spending  
• Line Chart for daily spending trend  
• Monthly comparison chart  

Widgets:

• Last 5 transactions  
• Budget progress bars  

--------------------------------

2. BUDGET LIMIT SYSTEM

Allow users to set monthly budget limits per category

Example:
Food Budget = 3000 THB

Features:
• Budget progress bar  
• Warning at 80% usage  
• Alert when budget exceeded  

Display in dashboard and reports.

• Yearly
